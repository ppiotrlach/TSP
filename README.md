# TSP

implemented:
- brute force <br>
- branch and bound least cost <br>
- simmulated annealing <br>

running: 
```console
./tsp arg1 arg2
```

arg1 = instance name with extension f.e m6.atsp <br>
arg2 = methodname: <br>
/t br = bruteforce <br>
/t bnb = branchandbound <br>
/t everything else runs simmulated annealing <br>
